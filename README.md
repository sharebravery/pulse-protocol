# Pulse Protocol — Retired

This repository is no longer part of the Pulse execution path.

Pulse now uses Markdown documents as the content handoff format:

```text
pulse-tasks
    ↓
pulse-agents
    ↓
Markdown
    ↓
pulse-relay
    ↓
email delivery
```

There is no shared JSON contract, generated schema, receipt, result model, or published package.

The repository remains only as a historical marker so old links do not silently point to a different implementation. Do not add new runtime code here.
